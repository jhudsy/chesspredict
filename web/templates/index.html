About

  • This website uses a machine learning model to predict the ratings of both players based on one lichess game, effectively "guessing their elo".
  • The current model (as of 7 January 2025) was trained on around 800,000 publicly accessible 5+0 games from lichess.
  • The system is running on a very low powered server. It may easily get overwhelmed, if things aren't working at the moment, please try again later.
  • If you have any problems, please open an issue on the project's github.
  • Source code is also available on the github site.
